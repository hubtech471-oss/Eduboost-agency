# Eduboost Marketing Agency Website

A premium creamy marketing agency website built with pure HTML, CSS & JavaScript.

## 🚀 GitHub Pages par Deploy Kaise Karein

### Step 1 — Repository Banao
1. GitHub.com par jao → **New Repository**
2. Repository name do: `eduboost-website`
3. **Public** select karo
4. **Add README** check karo
5. **Create Repository** click karo

### Step 2 — Files Upload Karo
1. Repository mein jao → **Add file** → **Upload files**
2. `index.html` drag & drop karo
3. **Commit changes** click karo

### Step 3 — GitHub Pages Enable Karo
1. Repository → **Settings** tab
2. Left sidebar mein **Pages** click karo
3. Source: **Deploy from a branch**
4. Branch: **main** → folder: **/ (root)**
5. **Save** karo

### Step 4 — Live Website 🎉
- 2-3 minute baad tera website live hoga
- URL milega: `https://yourusername.github.io/eduboost-website`

## ✏️ Customization
`index.html` file mein ye cheezein change kar:
- **Line ~280** → Apna phone number
- **Line ~284** → Apna email
- **Line ~288** → Apna city/address
- **Line ~215** → Stats (clients, ROI)
- **Colors** → CSS variables at top (`:root` section)

## 🛠️ Tech Used
- Pure HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts (Playfair Display + DM Sans)
- No frameworks, no dependencies, no backend needed
